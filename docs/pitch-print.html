The Problem
SAP training is expensive, inaccessible, and disconnected from real consulting work
  • 💸 $3,000–$8,000 per seat for SAP training licenses. IBMers on the bench cannot access hands-on practice without significant budget approval.
  • 🔒 Sandbox environments require infrastructure setup, system administration, and client or IBM IT approval — often unavailable during bench periods.
  • 📖 E-learning platforms teach button-clicking workflows. They do not train the cross-module investigation reasoning that clients pay IBM consultants to apply.
  • 📉 Skill decay on the bench means consultants arrive at engagements under-prepared, increasing ramp-up cost and client satisfaction risk.
The real gap: SAP consulting is not about knowing which screen to open — it is about understanding why the system behaved that way across FI, CO, SD, and MM. That reasoning cannot be trained by passive e-learning.

Solution: What We Built
IBM Bob: SAP SkillBridge Sandbox — v1.1
2 / 3
🖥 SAP Fiori Interface

Shell bar, 3-level SAP menu tree, transaction search field, selection screens, result grids, status bar — visually identical to SAP S/4HANA Fiori. The consultant navigates the menu to find transactions, exactly as on a real engagement.

🎯 Interactive Scenario Engine

5 business incidents with 7–8 investigation steps each. SAP terminal output, multiple-choice questions with detailed right/wrong explanations, completion score, and progress persistence via localStorage.

⚙️ Professional Mode

Real SAP workflow: Selection Screen → Execute (F8) → Result Grid. No coaching, no hints — exactly like the real system. Used for interview preparation and timed practice.

🤖 Bob Coaching Console

AI coaching layer providing real-time contextual guidance — describes the business problem, points to the relevant menu area, and explains findings in stakeholder language. Designed to evolve into a watsonx conversational layer.

🗺 Transaction Roadmap

Visual reference page: all 18 transactions by module and sub-area, with full descriptions, process flow strips (O2C, P2P, CO variance), and scenario tags showing where each transaction is used in practice.

📦 Zero Infrastructure

One Python file. No npm, no build pipeline, no SAP license, no cloud account. Runs on any laptop with Python 3. Open source on GitHub. Any IBMer on the bench can clone and start practicing in under 60 seconds.
